<h1>Type Diagrams</h1>



<h2> Visual Studio Viewer</h2>
<p> If you install ReSharper to Visual Studio, you can right-click on any file or folder, and click <b>Show Type dependency Diagram</b>.</p>
<p> Using the viewer let's you see the relationships between classes, the dependencies between them, and, if you hover over a box you will see the usages of that type too.</p>
